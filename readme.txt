在結帳時高效拆分冷凍和常溫商品，使用清晰標籤、自動和手動拆單選項、透明配送費用和智能推薦系統，以提升物流和顧客體驗。


== Description ==

在電商平台上，合理地拆分冷凍與常溫商品的訂單對於提升物流運營效率和控制成本至關重要。以下是一些具體的設計建議，能夠在結帳階段引導顧客拆單，並優化整體用戶體驗：

1.商品分類標籤：在商品頁面和購物車中，使用不同顏色或圖標清晰地標示冷凍商品和常溫商品。

2.購物車提醒：當購物車中同時存在冷凍和常溫商品時，顯示提示信息，告知顧客這些商品需要分開配送。

3.自動拆單建議：在結帳頁面自動將訂單拆分為冷凍商品和常溫商品的兩個訂單，並展示分開配送的優勢，如更快的配送速度和食品新鮮度的保證。

4.手動拆單選擇：提供顧客自行選擇是否拆單的選項，並清楚說明各選項的影響，例如不同的配送時間或費用。

5.配送費用透明化：分別顯示冷凍和常溫商品的配送費用，讓顧客清楚了解拆單的實際成本。明確告知顧客在達到一定金額時可以享受的免費配送政策，鼓勵合理拆單並增加銷售額。

6.簡化拆單流程：確保拆單操作簡單易懂，不增加顧客的購物負擔。對於拆單後的訂單，提供一鍵結帳功能，減少多次操作。

7.智能推薦系統：利用算法分析顧客的購買行為和偏好，自動推薦相關的常溫或冷凍商品，促使顧客一次性完成拆單購物。

8.後端系統整合：確保後端物流系統能夠高效處理拆單後的訂單，保持庫存更新、包裝和配送的流暢性。

這些設計策略不僅可以幫助電商平台提升運營效率，還能增強顧客的購物體驗，最終促進銷售和客戶滿意度。

This plugin allows customers to split their orders themselves at the checkout screen based on whether they prefer frozen or regular temperature shipping. Customers can divide the orders on their own, providing greater convenience for backend operators in managing inventory.


== Installation ==
1. 安裝 woocommerce。
2. 下載安裝本插件。
3. 插件會在運送方式加入冷凍宅配、常溫宅配等選項
4. 在商品編輯頁面冷凍商品在運送選單選擇“冷凍宅配”;反之選擇“常溫宅配”。
5. 消費者依據畫面顯示進行拆單結帳。
